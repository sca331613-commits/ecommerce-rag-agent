# 售后保障规则

---

## 第一章 消费者保障服务体系总则

### 第1条 服务宗旨

为保障消费者合法权益，规范平台经营秩序，卖家须自愿签署《消费者保障服务协议》并足额缴纳保证金后方可加入消费者保障服务体系。本体系分为**基础服务（必选）**与**特色服务（可选）**两类。

### 第2条 基础服务（必选）

所有入驻卖家均须履行的保障义务：

| 编号 | 服务项目 | 内容说明 |
|------|----------|----------|
| 2.1 | 商品如实描述 | 卖家须对商品的质量、成色、瑕疵、材质、规格等信息进行真实、准确、完整的描述，不得隐瞒或虚假宣传 |
| 2.2 | 七天无理由退货 | 买家签收商品后7天内可申请无理由退货 |
| 2.3 | 法定三包服务 | 依《消费者权益保护法》及国家三包规定提供修理、更换、退货服务 |
| 2.4 | 售后服务义务 | 卖家须对商品售后问题及时响应、妥善处理 |

### 第3条 特色服务（可选）

卖家可根据经营品类自愿加入以下增值保障服务：

| 编号 | 服务项目 | 赔付/保障内容 |
|------|----------|---------------|
| 3.1 | 假一赔三 | 确认售假后，退回货款并额外赔偿商品价款的2倍，总计退一赔三 |
| 3.2 | 假一赔四 | 确认售假后，赔付实际成交金额的4倍，无上限 |
| 3.3 | 正品保障 | 卖家承诺所售商品为品牌正品 |
| 3.4 | 破损包退 | 运输途中破损由卖家承担免费退换货责任 |
| 3.5 | 运费险 | 买家退货时由保险公司补偿退货运费 |
| 3.6 | 数码30天维修 | 数码家电类商品签收后30天内提供免费维修服务 |
| 3.7 | 闪电发货 | 虚拟商品未在规定时效内发货，买家可申请赔偿 |
| 3.8 | 退货承诺 | 卖家自行设置长于7天的无理由退货期（如15天、30天） |
| 3.9 | 假一赔十 | 特定品类（如名优白酒）适用更高赔偿标准 |

## 第二章 假一赔四规则

### 第4条 适用范围

"假一赔四"为卖家自愿加入的特色服务。仅当商品详情页**明确标注"支持假一赔四"**字样时，该保障规则才对该笔交易生效。

### 第5条 赔付标准

| 项目 | 标准 |
|------|------|
| 赔付倍数 | 实际成交金额的 **4倍** |
| 赔付上限 | **无上限** |
| 赔付示例 | 商品成交价2万元，鉴定为假货后赔付 **8万元** |
| 法律依据 | 《消费者权益保护法》第55条法定支持三倍赔偿；四倍赔偿属**平台规则+商家承诺**的额外保障 |

### 第6条 申请条件

出现以下情形之一，经平台核实后可启动赔付：

| 编号 | 情形 | 具体表现 |
|------|------|----------|
| 6.1 | 仿冒品牌商品 | 商标名称拼写模仿知名品牌、使用未获授权的知名品牌标识等 |
| 6.2 | 虚假原产地或认证 | 国产商品冒充进口、伪造CE认证标志、伪造3C认证标志等 |
| 6.3 | 未取得合法资质 | 食品无SC编号、化妆品未完成备案、电器无3C认证 |
| 6.4 | 包装或宣传信息虚假 | 宣传材质与实际严重不符、夸大功效、虚构材质成分 |

### 第7条 操作流程

| 步骤 | 操作事项 | 时间/说明 |
|------|----------|-----------|
| 第1步 | 确认标识：下单前核实商品详情页有"支持假一赔四"字样 | 须**截图保存**作为凭证 |
| 第2步 | 收货存证：开箱全程录制视频（一镜到底不间断） | 重点拍摄外包装、标签、商标、生产批号 |
| 第3步 | 辨别真伪：通过国家知识产权局、药监局等官网核验商标、备案号、3C认证号 | 取得官方鉴定结论或查询结果 |
| 第4步 | 提交投诉：订单详情页→售后→投诉商家→选择"商品涉嫌假货" | 上传全部证据材料 |
| 第5步 | 等待赔付：平台初步审核 | **3个工作日**内出具审核结论；确认假货后赔付款自动打入支付宝账户 |

### 第8条 2025年"平台鉴别"服务补充规定

| 项目 | 内容 |
|------|------|
| 覆盖品牌 | 爱马仕、路易威登、Gentle Monster、Human Made 等 **300余个** 奢侈品牌及潮牌 |
| 覆盖品类 | 服装、鞋帽、箱包、配饰等全品类 |
| 合作机构 | 中检集团（CCIC）、中溯检验检测认证中心等国家级权威鉴别机构 |
| 服务流程 | 买家拍下带"平台鉴别"标签的商品 → 商品先送鉴别机构 → 鉴别通过后出具具法律效力的鉴别证书 → 再寄送买家 |
| 赔付兜底 | 若平台鉴别结论与商品实际真伪不符，买家可获高达**4倍**商品价格的赔付 |
| 试运行数据 | 2025年6月试运行以来，售假投诉率下降 **82%**，仅 **0.3%** 的送检商品被鉴定为非正品 |

## 第三章 七天无理由退货规则

### 第9条 时间计算规则

| 环节 | 起算时点 | 截止时限 |
|------|----------|----------|
| 发起退货申请 | 物流信息显示**签收当日次日零时**起 | 满 **168小时（7天）**前 |
| 卖家提供退货地址 | 收到买家退货通知时起 | 不超过系统设定的超时时间 |
| 买家寄回商品 | 获得卖家退货地址时起 | 不超过系统设定的超时时间 |
| 卖家确认退款 | 收到退货并核验无误时起 | 通常 **1至3个工作日**到账 |

### 第10条 不支持七天无理由退货的商品清单

#### 10.1 绝对不支持类（8类）

| 序号 | 商品类别 | 具体范围 |
|------|----------|----------|
| 1 | 定制类商品 | 个性化定制、按需设计服务等 |
| 2 | 鲜活易腐类 | 鲜花绿植、水产肉类、新鲜蔬果、宠物、速冻食品等 |
| 3 | 数字化商品 | 网络游戏虚拟商品、话费充值、数字阅读、电子凭证、音像制品等 |
| 4 | 报纸期刊 | 订阅类报纸、期刊 |
| 5 | 服务类商品 | 本地生活服务、代购服务等 |
| 6 | 个人闲置物品 | 个人卖家出售的闲置二手物品 |
| 7 | 二手及竞拍商品 | 二手商品、拍卖渠道所得商品 |
| 8 | 人体用药 | 各类人体服用药品 |

#### 10.2 卖家可选支持类（2类）

| 序号 | 商品类别 | 不支持理由 |
|------|----------|------------|
| 9 | 激活/试用后价值贬损较大类 | 智能设备、商用厨电、黄金饰品等，一经激活或试用即大幅贬值 |
| 10 | 拆封后影响安全或易变质类 | 食品、保健品、贴身衣物、成人用品、酒水等，拆封后难以保证卫生安全 |

### 第11条 商品"完好"标准

| 商品类别 | 完好判定标准 | 不可为事项 |
|----------|--------------|------------|
| 通用标准 | 商品无受损、无受污；配件齐全（吊牌、说明书、三包卡等）；外包装完整 | 不得丢失配件、损坏包装 |
| 食品/保健品/化妆品 | 一次性密封外包装 **未拆封** | 不得拆开密封包装 |
| 图书/音像制品 | 一次性密封外包装 **未拆封**；无折痕、水渍、字迹 | 不得拆封、不得污损 |
| 内衣/泳衣 | 裆部保护贴 **不可撕下**；防盗扣不可剪断 | 不得撕除保护贴、不得剪防盗扣 |
| 运动鞋/服装 | 防盗扣不可剪断；**无穿戴痕迹**、无洗涤 | 不得拆防盗扣、不得穿戴外出、不得水洗 |
| 电子产品 | **未激活**、无个人数据痕迹 | 不得开机激活、不得导入个人数据 |
| 大家电 | 可通电调试，但**不得安装使用** | 空调不得上墙、洗衣机不得过水 |
| 手办/模型 | 内包装胶条、塑封 **不可拆** | 不得撕除内包装封条 |

### 第12条 运费承担规则

| 场景 | 发货运费承担方 | 退货运费承担方 |
|------|---------------|----------------|
| 商品包邮，买家无理由退货 | 卖家（已包邮） | **买家** |
| 商品未包邮，买家无理由退货 | **买家** | **买家** |
| 商品包邮，部分退货后仍满足包邮条件 | 按比例分摊 | **买家** |
| 商品存在质量问题/与描述不符 | **卖家** | **卖家** |

### 第13条 退款金额计算

| 退款项目 | 计算方式 |
|----------|----------|
| 商品价款 | 按实际支付金额全额退还 |
| 买家使用优惠券/红包 | 优惠部分不折算现金退还，退回等额优惠券（以平台规则为准） |
| 买家使用积分/金币 | 退还对应积分/金币 |
| 卖家发货费用 | 包邮商品退回时买家已承担的退货运费由买家自行承担（质量问题的除外） |

## 第四章 假一赔三规则

### 第14条 基本定义

假一赔三为卖家自愿加入的特色服务。买家购买带"假一赔三"标识的商品，经确认属售假行为后，可获得以下赔付：

| 赔付构成 | 金额说明 |
|----------|----------|
| 退还货款 | 退回买家实际支付的商品价款（即"退一"） |
| 赔偿金 | 额外赔偿商品价款的 **2倍** |
| 合计 | **退一赔三**（合计为商品价款的3倍） |
| 物流费用 | 卖家承担维权所涉商品的**全部物流费用** |

### 第15条 与"假一赔四"的关系

| 对比维度 | 假一赔三 | 假一赔四 |
|----------|----------|----------|
| 赔付倍数 | 退还价款+赔偿2倍=合计3倍 | 赔付实际成交金额的4倍 |
| 上限 | 以商品价款为基数 | 无上限 |
| 法律依据 | 《消费者权益保护法》第55条法定标准 | 平台规则+商家额外承诺 |
| 适用前提 | 商品页标注"假一赔三" | 商品页标注"假一赔四" |

## 第五章 附则

### 第16条 规则效力

本规则自公示之日起生效。平台保留根据法律法规变化及业务需要对规则进行修订的权利，修订后的规则将于公示期满后生效。

### 第17条 争议解决

买卖双方因售后保障事宜发生争议的，应先行协商解决；协商不成的，可向平台投诉或依法向有管辖权的人民法院提起诉讼。

### 第18条 术语释义

| 术语 | 释义 |
|------|------|
| 签收日 | 物流信息系统中记录的商品签收日期 |
| 实际成交金额 | 买家实际支付给卖家的商品价款（不含运费、税费等） |
| 一镜到底 | 自开箱起至商品检查完毕，全程不间断的连续录像 |
| 完好标准 | 商品保持原有品质、功能，商品本身及配件、商标标识齐全，并保持原包装状态 |
