# 淘宝网保证金管理规范

> **规范名称**：《淘宝网保证金管理规范》
> **首次生效**：2021年9月30日
> **最新修订**：2025年3月10日（2025年3月4日起实施）
> **官方原文**：https://jianghu.taobao.com/detail/47301_58665605

---

## 第一章 总则

### 第1条 制定目的

为维护淘宝网平台交易秩序，保障消费者合法权益，规范卖家经营行为，依据《淘宝平台服务协议》及相关法律法规，制定本规范。

### 第2条 适用范围

本规范适用于所有在淘宝网平台开设店铺并开展经营活动的卖家，包括个人店铺、企业店铺及各类品牌旗舰店。

### 第3条 保证金定义

保证金是指卖家为履行消费者保障义务而向平台缴存的资金保证。保证金以冻结形式存放于卖家绑定的支付宝账户内，用于担保交易过程中可能出现的赔付责任。

---

## 第二章 保证金额度构成

### 第4条 总额度计算公式

保证金额度由以下三部分动态叠加构成：

```
总额度 = 基础额度 + 风险额度 + 特殊额度
```

每月依据近30天经营数据进行动态调整。

### 第5条 基础额度

基础额度根据店铺近30天成交金额最高的类目，对应全店成交阶梯档位确定。

| 类目类型 | 参考基础额度 |
|----------|-------------|
| 普通服装、日用百货等一般类目 | 约 1,000 元 |
| 数码3C、家电等类目 | 5,000 元 ~ 10,000 元 |
| 食品、化妆品、虚拟商品等特殊类目 | 以卖家中心后台提示为准（通常更高） |
| 无成交新店铺 | 0 元（暂无需缴纳） |

### 第6条 风险额度

卖家因违规行为（包括但不限于售假、虚假宣传、违背承诺、描述不符等）被平台综合评估后，系统即时提升相应风险额度。风险额度自违规风险完全消除之日起自动归零。

### 第7条 特殊额度

卖家因准入特定产品类目或特殊市场（包括但不限于二手奢侈品、珠宝玉石、虚拟商品、游戏代练等）而需额外缴存的额度，按对应准入规则执行。

### 第8条 动态调额规则

| 调额节点 | 规则说明 |
|----------|----------|
| 每月6日 | 依据最近30天成交情况重新计算基础额度 |
| 当月有欠费记录 | 本月额度不低于上月额度 |
| 当月无欠费记录 | 本月额度不高于上月额度 |
| 违规触发 | 风险额度即时上调，不受月度周期限制 |

## 第三章 缴纳方式与时限

### 第9条 缴纳方式

卖家可选择以下三种方式之一完成保证金缴纳：

| 缴纳方式 | 说明 | 费用参考 |
|----------|------|----------|
| 现金缴纳 | 通过支付宝一次性缴存，资金冻结在保证金账户 | 全额冻结 |
| 保证金保险（履约险） | 购买保险替代现金缴存 | 约 30 元抵 1,000 元；约 1,500 元/年抵 50,000 元 |
| 账期保障 | 交易款项延长账期（通常15天）作为保障替代 | 无需实际缴存资金 |

> 注意：保证金保险通常不对新店铺立即开放，需经营一段时间并达到一定等级后方可购买。

### 第10条 缴纳时限

| 情形 | 时限要求 |
|------|----------|
| 额度确定后 | 须在 **72小时内** 完成足额缴存 |
| 新店铺 | 开店后可先发布商品，**首个成交日次日**起方需缴纳基础保证金 |
| 保证金余额不足触发补缴 | 系统自动扣款失败后，须在 **72小时内** 自行补足 |

### 第11条 操作路径

卖家通过以下路径完成缴纳：

```
登录卖家中心 → 店铺管理 → 保证金管理 → 选择缴纳方式并支付
```

### 第12条 未按时缴纳的后果

未在规定时限内足额缴纳保证金的，平台将按以下顺序逐步升级管控措施：

| 阶段 | 管控措施 |
|------|----------|
| 第一阶段 | 延长交易账期 |
| 第二阶段 | 限制参加营销活动 |
| 第三阶段 | 屏蔽店铺 |
| 第四阶段 | 监管账户 |
| 第五阶段 | 支付账户管控 |

## 第四章 保证金划扣规则

### 第13条 平台直接划扣情形

在以下情形中，平台有权从卖家保证金中直接划扣相应金额：

| 划扣情形 | 具体说明 |
|----------|----------|
| 交易争议判定卖家责任 | 平台划扣用于退款或赔付消费者 |
| 违规行为处置 | 售假、虚假宣传、违背承诺等违规行为，按情节严重程度划扣 |
| 第三方投诉判定责任 | 经平台判定责任归属后执行划扣 |

### 第14条 关联店铺连带责任

同一卖家或同一实际经营者在平台开设多个店铺的，当其中一个店铺的保证金不足以履行赔付义务时，平台有权从其他关联店铺的保证金中划扣补足。卖家应做好跨店铺资金规划，防止连带划扣影响正常经营。

### 第15条 自动补缴机制

保证金账户余额低于应缴额度时，系统将自动从绑定的支付宝账户扣款补足。若自动扣款失败，卖家须在 **72小时内** 自行补足差额；逾期未补足的，平台将启动相应的管控措施。

## 第五章 保证金退还规则

### 第16条 基础额度退还条件

申请退还基础额度保证金须同时满足以下 **全部6项** 条件：

| 序号 | 条件 |
|------|------|
| 1 | 没有进行中的交易记录 |
| 2 | 没有进行中的退款记录 |
| 3 | 没有进行中的投诉记录 |
| 4 | 没有交易成功后 **30天内** 的订单（即最后一笔交易成功须满30天） |
| 5 | 没有正在处理中的欠费记录 |
| 6 | 没有未完结的违规限制解冻记录 |

### 第17条 风险额度退还条件

违规风险完全消除后，风险额度自动调整为0。退出前须确认违规处理期已满：

| 违规类型 | 处理期要求 |
|----------|------------|
| 一般违规 | 违规处理完结即可 |
| 严重违规 | 可能须满 **12个月** |

### 第18条 特殊额度退还条件

按相应产品类目或特殊市场的准入退出规则执行，须先退出相应市场方可申请退还。

### 第19条 退还操作流程

| 步骤 | 操作内容 |
|------|----------|
| Step 1 | 登录卖家中心 |
| Step 2 | 进入「消费者保障服务」/「保证金管理」页面 |
| Step 3 | 点击「申请退还保证金」 |
| Step 4 | 填写退款原因 |
| Step 5 | 平台审核（官方承诺 **3 ～ 5 个工作日**，最快 **2 天** 完成） |
| Step 6 | 审核通过后，保证金退回至绑定的支付宝账户（到账时间 **1 ～ 7 天**） |

### 第20条 额度调整自动退回规则

| 情形 | 退回方式 |
|------|----------|
| 保证金额度调整为 **0** | 资金自动退回至可提现余额，卖家自行操作提现至支付宝账户 |
| 额度部分下调 | 超出下调后额度的部分可单独申请退回 |

### 第21条 退还注意事项

| 注意事项 | 具体要求 |
|----------|----------|
| 店铺状态 | 须完成店铺注销或关闭，不得留有空壳店铺 |
| 等待期 | 部分情况下店铺关闭后须等待 **90天** 方可申请退还 |
| 违规记录 | 刷单、售假等历史违规须全部处理完毕 |
| 欠费清缴 | 广告费、软件服务费、平台使用费等须全部结清 |
| 保证金存续期限 | 部分情况下须保证金交满 **12个月** |
| 审核期间行为 | 审核期间不得操作店铺，否则可能被判定为"异常经营"导致申请被驳回 |

## 第六章 企业店铺及旗舰店保证金

### 第22条 企业店铺/旗舰店保证金标准

| 店铺类型 | 保证金额度范围 |
|----------|---------------|
| 一般企业店铺 | 5,000 元 ~ 50,000 元 |
| 品牌旗舰店 | 10,000 元 ~ 100,000 元（视类目和品牌授权情况而定） |

企业店铺及旗舰店的最终保证金额度以卖家中心后台实时显示为准。

---

## 第七章 监督管理

### 第23条 额度变动通知

平台在以下情况下将通过站内信或短信方式通知卖家：

| 通知场景 | 通知时限 |
|----------|----------|
| 基础额度调整 | 每月6日调整后 |
| 风险额度触发 | 违规判定生效后即时通知 |
| 余额不足预警 | 保证金余额低于应缴额度时 |
| 划扣执行 | 划扣完成后即时通知 |

### 第24条 数据查询

卖家可通过以下路径实时查询保证金状态：

```
卖家中心 → 店铺管理 → 保证金管理
```

该页面显示当前额度的三部分构成明细、历史划扣记录、缴纳方式及余额信息。

### 第25条 卖家义务

1. 卖家应当如实提供经营信息，不得通过虚假信息规避保证金额度评估。
2. 卖家应当确保绑定支付宝账户资金充足，以保障自动补缴机制正常运行。
3. 关联店铺的卖家应当统筹管理各店铺保证金，防范连带划扣风险。
4. 卖家应当至少每月6日后登录查看一次额度变动，及时完成补缴。

---

## 第八章 附则

### 第26条 规范修订

平台有权根据法律法规变化、运营策略调整及市场环境变化对本规范进行修订。修订后的规范将在平台公告栏公示，公示期满后正式生效。

### 第27条 最终解释权

本规范由淘宝网负责解释。具体保证金额度以卖家中心实时显示为准，本文档为规则框架参考。

### 第28条 生效日期

本规范自 **2025年3月4日** 起正式实施，原《淘宝网保证金管理规范》（2021年9月30日版）同时废止。

## 附录：2025年修订要点速览

| 序号 | 修订要点 |
|------|----------|
| 1 | 动态调整机制细化：每月6日根据近30天成交重新计算额度，规则更透明 |
| 2 | 关联店铺连带责任明确化：跨店铺划扣规则进一步细化，遏制批量多店铺扰乱市场 |
| 3 | 保证金履约险推广：高额保证金可换购年付保险（如1,500元/年抵50,000元），缓解卖家现金流压力 |
| 4 | 新店铺门槛降低：开店后可先发布商品，产生首笔订单后方需缴纳基础保证金 |
