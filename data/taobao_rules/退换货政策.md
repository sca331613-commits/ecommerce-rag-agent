# 电商平台退换货政策规则

> **参考依据**：淘宝网2025年最新版退换货政策规则，综合自淘宝官方规则中心及多个权威解读来源。

---

## 第一章 总则

### 第1条 规则版本与生效时间

| 规则文件 | 公示时间 | 生效时间 |
|----------|----------|----------|
| 《淘宝网七天无理由退货规范》 | 2025年7月21日 | **2025年7月28日** |
| 《淘宝平台争议处理规则》 | — | **2025年8月15日** |

### 第2条 七天无理由退货定义

买家购买带有 **"7天无理由退货"服务标识**的商品后，自签收商品之日起7天内，对符合完好标准的商品，可向卖家发起退货申请。该标识展示在商品详情页的"服务保障栏"。

### 第3条 适用范围

本规则适用于平台内所有带有"7天无理由退货"服务标识的商品交易。部分特殊类目商品依法或依规则不适用七天无理由退货，详见第六章规定。

---

## 第二章 时间限制与计算规则

### 第4条 七天计算方式

| 计算要素 | 具体规定 |
|----------|----------|
| **起算点** | 自物流显示签收的**次日零时**起计算 |
| **时长** | 满 **168小时** 为7天 |
| **最晚发起时间** | 必须在168小时内提交退货申请，以系统记录时间为准 |

### 第5条 退款时效

卖家确认收货后，货款在 **1-3个工作日**内退回买家原支付方式。

### 第6条 卖家响应时效

卖家须在系统限定时间内提供退货地址、联系人、联系电话。若卖家超时未提供，系统可自动介入处理。

## 第三章 申请条件

### 第7条 商品完好通用标准

商品必须同时满足以下全部条件：

1. **保持原有品质、功能**，配件齐全（吊牌、说明书、三包卡等）；
2. **无受损、受污、刮开防伪、激活（授权）等情形**；
3. **无难以恢复原状的外观类使用痕迹**、不合理的个人数据使用痕迹；
4. 买家基于**查验需要**打开商品包装，或为确认品质、功能进行**合理调试**，不影响完好；
5. **超出查验和确认需要**而使用商品，导致价值贬损较大的，视为不完好。

### 第8条 包装完好标准

#### 第8.1款 一般商品包装

- 商品原包装**未缺失**，包装整体结构完整；
- 运输或拆卸过程中产生的**难以避免的形态变化**（适度变形、污渍、局部破损等），**不认定**为影响包装完整。

#### 第8.2款 特殊商品包装

- 若包装客观上具有**防伪性、唯一性、收藏纪念价值**等特殊附加价值，卖家可在**合理范围内**提出特定的包装保护要求；
- 须**事前显著明示**（商品页醒目位置提示、旺旺告知、包裹粘贴提醒）；
- 买家未按明示要求保护且包装损坏足以造成价值贬损的，可认定商品价值贬损；
- 若卖家**未事前显著明示**，则按一般商品包装标准处理。

#### 第8.3款 包装界定说明

"商品包装"指**厂商实施的出厂标准包装**，不含卖家或物流另行二次封装。卖家使用厂商原包装发货，或消费者签收时外包装已破损，卖家仍要求退货时外包装不得破损的，视为不合理要求。

## 第四章 各类目特殊完好标准

### 第9条 类目特殊要求明细表（2025年7月28日版）

| 类目 | 特殊完好标准 |
|------|-------------|
| **书籍/杂志/报纸** | 一次性密封外包装未拆封；无折痕、水渍、污损、字迹、阅读痕迹 |
| **运动鞋/女鞋/男鞋/箱包** | 防盗扣/防损扣不可剪掉（不影响合理检视前提下）；无显著穿戴、无洗涤、无化妆品痕迹 |
| **泳衣** | 裆部保护贴不得撕下 |
| **女士内衣/男士内衣** | 一次性密封外包装未拆封 |
| **模型/手办/兵人/BJD** | 内包装胶条、塑封不可拆 |
| **笔记本电脑/DIY电脑/平板** | 系统未激活；进网许可证未损坏；**未有浸液**（新增） |
| **游戏软件** | 外包装一次性密封未拆封 |
| **厨房电器** | 净（滤）水设备不得过水 |
| **净水器/空气净化器滤芯** | 经卖家事前显著明示后，一次性密封包装未拆封（新增） |
| **电子元器件** | 无源元件无难以恢复的使用痕迹；易耗品（导热硅脂/电子胶等）一次性密封包装未拆封（新增） |
| **二手数码** | 防撕标完好；商品无新增划痕破损；配附件齐全无损坏（新增） |
| **智能设备** | 进网许可证未损坏；未有浸液；系统未激活；智能眼镜/VR/MR/AR设备不支持7天无理由 |
| **珠宝/钻石/翡翠/文玩/邮币/字画/收藏** | 鉴定证书完整无显著污染；标签牌完整（文玩收藏类新增适用） |
| **大家电** | 液晶电视/空调可简单通电调试，不得上墙安装；洗衣机不得过水 |
| **食品（含保健食品）/化妆品/医疗器械/计生用品/个人护理用品/成人用品/婴儿尿片/图书/音像制品** | 若外包装被厂商以封条（签）或整体塑（密）封的，**不得拆封** |

## 第五章 退货流程

### 第10条 标准退货流程

#### 步骤1：发起申请

在淘宝APP/网页端 → 订单详情页 → 点击 **"申请售后" → "退货退款"** → 选择原因为 **"7天无理由退货"**。

#### 步骤2：卖家提供退货地址

卖家须在系统限定时间内提供：退货地址、联系人、联系电话。若卖家超时未提供，系统可自动介入处理。

#### 步骤3：寄回商品

买家按卖家提供的地址寄回商品，并**务必在系统中填写退货物流单号**。

> **重要提示**：寄回后必须填写物流单号，否则卖家可能以"没收到"为由拒绝退款。

#### 步骤4：卖家确认与退款

- 卖家确认收货后，货款在 **1-3个工作日**内退回买家原支付账户；
- 若卖家拒绝退款，买家可申请**淘宝客服介入**。

## 第六章 不支持七天无理由退货的商品

### 第11条 法定/规则强制不适用类目

| 类别 | 典型举例 |
|------|---------|
| **定制类商品** | 个性定制、设计服务 |
| **鲜活易腐类** | 鲜花、水产肉类、宠物、巧克力、月饼、速冻食品 |
| **数字化商品** | 网游、话费、数字阅读、电子凭证 |
| **报纸期刊** | 订阅类 |
| **服务类商品** | 本地服务、代购服务 |
| **个人闲置/二手/竞拍** | — |
| **人体用药** | — |
| **智能眼镜/VR/MR/AR设备** | 特殊完好标准中明确排除 |

### 第12条 卖家可选择支持（但可主动承诺支持7天无理由）

- 激活后价值贬损较大的商品（智能商品、商用厨电、黄金等）；
- 拆封后影响人身安全或品质的商品（食品、保健品、贴身用品、成人用品、酒水等）。

### 第13条 必须支持七天无理由的商品（标服类目）

服装服饰（除贴身内衣裤）、数码产品、家纺家居、母婴用品（除食品及贴身用品）、图书、珠宝钻石、家具、家电等。

## 第七章 运费承担规则

### 第14条 七天无理由退货运费（依据争议处理规则第五十三条）

| 情形 | 运费承担 |
|------|---------|
| **包邮商品** + 买家无理由退货 | 买家**仅承担退回运费** |
| **非包邮商品** + 买家无理由退货 | **来回运费均由买家承担** |
| 买家部分退货导致不再满足包邮条件 | 所有运费由买家承担 |
| 买家滥用会员权利（恶意退货） | 所有运费由买家承担 |

### 第15条 七天无理由换货运费（依据争议处理规则第五十四条）

支持七天无理由换货的商品，买家七天无理由换货的，由**买家承担换货所产生的所有运费**。

## 第八章 附则

### 第16条 争议处理

买卖双方就退换货产生争议的，优先适用《淘宝平台争议处理规则》。平台有权根据本规则及争议处理规则对争议作出判定并执行相应处理。

### 第17条 规则更新

平台有权根据法律法规变化及运营需要对本规则进行修订，修订后的规则将在公示期满后正式生效。买家与卖家应及时关注规则更新，继续使用平台服务即视为同意修订后的规则。

### 第18条 生效与解释

本规则自 **2025年7月28日**起生效（其中争议处理规则相关条款自 **2025年8月15日**起生效），平台对本规则拥有最终解释权。
